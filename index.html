🔧 Firebase Setup Required for Sync

To enable cross-device sync with Google login, you need to set up a free Firebase project:

  1. Go to console.firebase.google.com → Create a project
  2. Enable Google sign-in under Authentication → Sign-in method
  3. Create a Realtime Database (start in test mode)
  4. Go to Project Settings → Add web app → copy config
  5. Open this HTML file and replace the firebaseConfig values near the top

Until then, the planner works fully offline — data is saved locally in your browser.
